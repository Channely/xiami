2.项目启动
	|-- 本地：
		|-- sails lift 
	|-- 部署
		|-- forever start xiami.js
	|-- 配置mongodb数据
		|-- 新建xiami-dev数据库
		|-- 新建user
			|-- 密码: sa
			|-- 用户名: root

	|-- 根据人名去查询数据库, 发现了，数据直接从数据库中取，

	|-- 框架

	|-- 运行
